# Student Marks Management System

A lightweight command-line application written in **C** to manage student academic records. This project demonstrates file handling, structures, and basic data management.

## 🚀 Features
- **Persistent Storage**: Saves data to a binary file (`students.dat`).
- **Record Entry**: Add student details (ID, Name) and marks for multiple subjects.
- **Auto-Calculation**: Automatically calculates the total score for each student.
- **View Records**: Formatted tabular display of all stored student data.

## 🛠️ Installation & Usage

### Prerequisites
- A C compiler (GCC, Clang, or MSVC).

### Compilation
Open your terminal and run:
